# Student Analytics Performance System

A complete web application for managing student performance, attendance, and generating weekly reports.

## Features

### Students
- View scores and attendance
- See performance charts
- Get weekly performance reports

### Teachers
- Add/update student scores
- Mark attendance
- Generate weekly reports
- View all students

### Admin
- Manage users (create, edit, delete)
- View system statistics
- Monitor all activities

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Authentication**: JWT (JSON Web Tokens)
- **Real-time**: Socket.io
